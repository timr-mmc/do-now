## QUESTION TYPE SUGGESTIONS BY SHAPE

### Rectangle Questions (RECT_01-05):
1. Find the area
2. Find the perimeter
3. Find missing dimension given area
4. Find diagonal length (Pythagoras)
5. Scale factor problems
6. Real-world contexts (gardens, rooms, fields)

### Square Questions (SQUARE_01-04):
1. Find the area
2. Find the perimeter
3. Find side length given area/perimeter
4. Find diagonal length
5. Comparison with rectangles

### Triangle Questions (TRI_*):
1. Find the area
2. Find the perimeter
3. Use Pythagoras to find missing side
4. Verify Pythagorean triple
5. Compare different triangle types

### Parallelogram Questions (PARA_01-03):
1. Find the area (base × height)
2. Find the perimeter
3. Compare with rectangle of same dimensions
4. Find missing dimension

### Trapezium Questions (TRAP_01-03):
1. Find the area using formula: ½(a+b)h
2. Find missing parallel side
3. Find height given area
4. Compare with other quadrilaterals

### Circle Questions (CIRCLE_01-05):
1. Find circumference
2. Find area
3. Find radius/diameter given circumference or area
4. Semicircle and quarter circle problems
5. Real-world contexts (wheels, pipes, plates)

### Compound Shape Questions (COMP_01-05):
1. Find total area (add/subtract components)
2. Find perimeter of outer edge
3. Identify component shapes
4. Calculate materials needed
5. Real-world applications

### Cuboid Questions (CUBOID_01-04):
1. Find the volume
2. Find the surface area
3. Find specific face areas
4. Capacity/litres conversion
5. Packing/fitting problems

### Cube Questions (CUBE_01-02):
1. Find the volume
2. Find the surface area
3. Find edge length given volume
4. Comparison with cuboids

### Prism Questions (TRI_PRISM_01-03):
1. Find the volume (cross-sectional area × length)
2. Find cross-sectional area
3. Find surface area
4. Real-world contexts (beams, tents)

### Cylinder Questions (CYLINDER_01-03):
1. Find the volume (πr²h)
2. Find the surface area (2πr² + 2πrh)
3. Find curved surface area
4. Find dimension given volume
5. Real-world contexts (cans, tanks, pipes)

## TIPS FOR CREATING QUESTIONS

- **Progressive Difficulty**: Start with "find the area/volume", then move to "find missing dimension", then to "compare" or "real-world application"
- **Mixed Skills**: Combine measurement with algebra (e.g., "if the area is 96 cm² and length is 12 cm, find width")
- **Unit Conversion**: Include questions requiring mm↔cm↔m conversions
- **Exact vs Approximate**: For circles, ask for both exact (with π) and approximate answers
- **Context**: Add real-world scenarios (gardens, rooms, containers, sports fields)
- **Comparison**: Use pairs of shapes to compare areas, volumes, efficiency
- **Multi-step**: Combine operations (e.g., find area of compound shape, then cost per m²)
